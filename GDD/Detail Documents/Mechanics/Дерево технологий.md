# Описание

Исследования требуются для улучшения существующих технологий и условий жизни колонистов. Для того, чтобы начать проводить исследования, нужно создать исследовательский стол, затем выбрать нужное исследование, открыв меню исследований. После того, как исследование выбрано, исследователь начнет им заниматься, генерируя очки исследований. Скорость исследования технологий напрямую зависит от уровня навыка [[Навыки#Умственный труд|Умственный труд]] у колониста-исследователя и его состояния [[Здоровье|здоровья]]. Для ускорения процесса рекомендуется использовать несколько столов с исследователями. Количество очков исследования зависит от технологического уровня колонии.

# Уровни технологий

## Примитивный уровень

В начале игры доступен только простой стол для исследований позволяющий изучать начальные технологии с малой эффективностью.

## Продвинутый уровень

После изучения технологии "Основы микроэлектроники" вам будет доступен высокотехнологичный стол для исследования с помощью которого можно изучать более продвинутые технологии "второго" уровня. Также ускоряет процесс исследований на 10%.

## Высший уровень

Технология "Мультианализаторы" позволит размещать рядом с исследовательскими столами [[Постройки|мультианализатор]] повышающий скорость исследований и открывающий последний уровень технологий оканчивающийся технологиями постройки космического корабля.

Скорость исследований не только зависит от [[Навыки|навыков]] и состояния [[Здоровье|здоровья]] учёного. Эффективность ещё можно повысить установив [[Постройки|стерильную плитку]]. Чистота помещения напрямую влияет на скорость исследования. Открыть постройку покрытия можно с помощью технологии.
