# Описание

Материалы - это ресурсы, используемые для строительства [[Постройки|построек]], изготовления [[предметы|предметов]], создания [[Броня|брони]] и [[Оружие|оружия]]. Каждый материал имеет свои характеристики, включая прочность, красоту, стоимость и доступность.

# Каменные блоки

## Блоки гранита

Один из самых прочных строительных материалов. Гранит имеет высокую прочность и огнестойкость, что делает его идеальным для оборонительных сооружений и стен.

| Характеристика | Значение |
| :------------- | :------- |
| Прочность      | 200      |
| Красота        | 0        |
| Стоимость      | 0.6      |
| Огнестойкость  | Полная   |

Гранит широко распространён в [[Горы|горах]] и подходит для строительства в любом [[Биомы|биоме]].

## Блоки известняка

Более мягкий камень, легче добывается но менее прочен чем гранит. Имеет светлый цвет, что может быть полезно для эстетических построек.

| Характеристика | Значение |
| :------------- | :------- |
| Прочность      | 150      |
| Красота        | 0        |
| Стоимость      | 0.6      |
| Огнестойкость  | Полная   |

Известняк подходит для ранних построек когда гранит ещё недоступен.

## Блоки мрамора

Декоративный камень с высокой красотой. Используется для создания красивых помещений, повышающих [[Нужды|настроение]] [[Колонисты|колонистов]].

| Характеристика | Значение |
| :------------- | :------- |
| Прочность      | 180      |
| Красота        | +4       |
| Стоимость      | 0.6      |
| Огнестойкость  | Полная   |

Мрамор рекомендуется использовать в спальнях и общих помещениях для бонуса к [[Красота|красоте]] окружения.

## Блоки песчаника

Наиболее распространённый и легко добываемый камень. Имеет среднюю прочность и подходит для большинства строительных задач.

| Характеристика | Значение |
| :------------- | :------- |
| Прочность      | 140      |
| Красота        | 0        |
| Стоимость      | 0.6      |
| Огнестойкость  | Полная   |

Песчаник идеален для ранних построек и временных сооружений.

## Блоки сланца

Прочный камень с нейтральными характеристиками. Сланец занимает промежуточное положение между гранитом и песчаником.

| Характеристика | Значение |
| :------------- | :------- |
| Прочность      | 160      |
| Красота        | 0        |
| Стоимость      | 0.6      |
| Огнестойкость  | Полная   |

Сланец подходит для строительства когда другие камни недоступны.

# Металлы

## Сталь

Основной строительный и производственный материал в игре. Сталь используется для создания [[Оружие|оружия]], [[Броня|брони]], механизмов и продвинутых построек.

| Характеристика | Значение |
| :------------- | :------- |
| Прочность      | 200      |
| Красота        | 0        |
| Стоимость      | 1.7      |
| Огнестойкость  | Низкая   |

Сталь получается из [[Ресурсы|руды]] через [[Постройки|плавильню]] или покупается у торговцев. Требуется изучение [[Дерево технологий|технологии]] "Электроплавка".

## Серебро

Драгоценный металл, используемый в основном для торговли и украшения. Серебро имеет высокую стоимость, но низкую прочность.

| Характеристика | Значение |
| :------------- | :------- |
| Прочность      | 140      |
| Красота        | +8       |
| Стоимость      | 17       |
| Огнестойкость  | Низкая   |

Серебро подходит для создания [[предметы|украшений]], [[Постройки|статуй]] и других декоративных элементов. Также используется как валюта в торговле с [[Фракции|фракциями]].

## Золото

Самый дорогой металл в игре. Золото используется  для декоративных целей и торговли и некоторой брони.

| Характеристика | Значение |
| :------------- | :------- |
| Прочность      | 140      |
| Красота        | +12      |
| Стоимость      | 34       |
| Огнестойкость  | Низкая   |

Золото повышает [[Красота|красоту]] помещений максимально, что даёт значительный бонус к [[Нужды|настроению]] [[Колонисты|колонистов]].

## Пласталь

Сверхпрочный сплав будущего. Пласталь используется для создания продвинутого [[Оружие|оружия]], [[Броня|брони]] и космических кораблей.

| Характеристика | Значение |
| :------------- | :------- |
| Прочность      | 400      |
| Красота        | 0        |
| Стоимость      | 25       |
| Огнестойкость  | Высокая  |

Пласталь нельзя создать в [[Колония|колонии]], только найти в руинах или купить у торговцев. Требуется для изучения продвинутых [[Дерево технологий|технологий]].

## Уран

Радиоактивный металл, используемый в ядерных реакторах и оружии. Уран опасен для [[Здоровье|здоровья]] при неправильном хранении.

| Характеристика | Значение |
| :------------- | :------- |
| Прочность      | 200      |
| Красота        | 0        |
| Стоимость      | 50       |
| Огнестойкость  | Высокая  |

Уран требует специальных условий хранения. Длительное нахождение рядом с ураном без защиты вызывает [[Болезни|болезни]] и радиационное отравление.

# Органические материалы

## Древесина

Основной ранний строительный материал. Древесина легко доступна но имеет низкую прочность и горючесть.

| Характеристика | Значение |
| :--- | :--- |
| Прочность | 100 |
| Красота | +2 |
| Стоимость | 0.7 |
| Огнестойкость | Низкая |

Древесина получается из [[Дерево технологий|деревьев]] через [[Работы#Садовод|садоводов]]. Подходит для ранней мебели и временных построек. Не рекомендуется для стен в пожароопасных зонах.

# Свойства материалов

## Прочность

Прочность определяет сколько урона может выдержать постройка из материала перед разрушением. Высокая прочность важна для оборонительных сооружений и стен.

## Красота

Красота материала влияет на [[Красота|красоту окружения]], что напрямую воздействует на [[Нужды|настроение]] [[Колонисты|колонистов]].

| Уровень красоты | Материалы |
| :--- | :--- |
| Отрицательная | Нет материалов |
| Нейтральная (0) | Гранит, известняк, песчаник, сланец, сталь, пласталь, уран |
| Положительная (+2) | Древесина |
| Высокая (+4) | Мрамор |
| Очень высокая (+8) | Серебро |
| Максимальная (+12) | Золото |

## Стоимость

Стоимость материала влияет на [[Богатство|богатство]] [[Колония|колонии]] и торговую ценность. Высокая стоимость повышает ожидания [[Колонисты|колонистов]].

## Огнестойкость

Огнестойкость определяет устойчивость материала к пожарам. Горючие материалы могут стать причиной распространения огня по базе.

# Применение материалов

## Строительство

Материалы используются для возведения стен, полов и других построек. Выбор материала зависит от назначения [[постройки]].
## Изготовление

Материалы требуются для создания [[предметы|предметов]], [[Броня|брони]] и [[Оружие|оружия]] на верстаках:

- Сталь - основное [[Оружие|оружие]], механизмы
- Пласталь - продвинутое [[Оружие|оружие]], космические компоненты
- Древесина - простая мебель, луки
- Серебро и золото - украшения, статуи

## Торговля

Некоторые материалы имеют высокую торговую ценность:

- Серебро - основная валюта торговли
- Золото - высокая стоимость, компактное хранение
